# Practical 6 – Django CRUD Project

This repository contains **Practical No. 6** from the FDP practical sessions.

## 📌 Aim

To create a basic **CRUD (Create, Read, Update, Delete) web application using Django framework**.

## ⚙️ Technologies Used

* Python
* Django
* SQLite
* HTML
* Visual Studio Code

## 📂 Project Structure

```
CRUD/
accounts/
manage.py
db.sqlite3
```

## 🚀 Features

* Create new records
* View records
* Update records
* Delete records

## ▶️ How to Run the Project

1. Install Django

```
pip install django
```

2. Run migrations

```
python manage.py migrate
```

3. Start the server

```
python manage.py runserver
```

4. Open in browser

```
http://127.0.0.1:8000/
```
